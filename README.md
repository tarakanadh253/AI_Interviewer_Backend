# OHG365 AI Interview Buddy

Welcome to the OHG365 AI Interview Buddy project.

## Project Description

AI-powered voice-based mock interviews designed to help you practice your tech interviews. Get instant feedback on your communication and technical skills.

## Technologies Used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Python (Django Backend)
- Supabase

## Setup Instructions

### Frontend
1. Navigate to the `frontend` directory.
2. Install dependencies: `npm install` or `pnpm install`
3. Run development server: `npm run dev` or `pnpm dev`

### Backend
1. Navigate to the `backend` directory.
2. Install requirements: `pip install -r requirements.txt`
3. Run server: `python manage.py runserver`
